# Claude Chat Service Client Adapter

This package provides a thin adapter that wraps the autogenerated client for `ai_chat_service`
and exposes the original `ai_chat_api.Client` interface so callers don't need to think about HTTP.

## Usage

1. Export the service OpenAPI spec:

   ```powershell
   uv run python -m scripts.export_ai_openapi
   ```

2. Generate the typed client package:

   ```powershell
   uv run openapi-python-client generate --path .\docs\ai_chat_service_openapi.json --output-path .\src\claude_chat_service_client --overwrite
   ```

3. Use the adapter:

   ```python
   from claude_chat_adapter import ServiceClaudeChat

   client = ServiceClaudeChat(base_url="http://localhost:8000", session_token="<jwt>")
   reply = client.send_message(prompt="Hello", user_id="user@example.com")
   print(reply.content)
   ```

The adapter takes care of setting the cookie-based session token and mapping the response to
`ai_chat_api.Message`.
